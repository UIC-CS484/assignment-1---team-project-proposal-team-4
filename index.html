RESTFul API:
API Name: Polygon.io Justification: Polygon.io is a verbose, fast stock API that provides real time & historical data on US stocks. Alongside that, it provides a javascript client library that is well documented. Other popular stock applications and websites utilize this API as well.

Insight and Information for end user:
The API we choose needs to be able to Provide real time data on the current price of stocks Provide detailed information on a stock With this information, weâ€™ll be able to provide insights on the stocks that the end-user may want to track or invest in. Alongside that, this information can be used to track the potential gains or losses the end-user has made. Our web app will allow users to visualize this data for multiple stocks on one graph allowing them to gain insight on patterns and trends. We will also allow visualization of different time periods of the same stock on a single chart to give users insight on potential annual patterns.

Data collected from authenticated user:
Sensitive User Information First and Last Name, DOB, Email, Phone Number Stocks they choose to track

Data tables:
USER -> fname (string), lname (string), dob (date), email (string), password (string), stocks they are watching (array), investment information (array)
Potential users:
Retail investors looking to gain more insight into their stock, crypto, and options investment strategies.

Goals of the site:
Allow users to visualize 2 or more stock price history charts over each other, to be able to visualize any trends between them.
Allow users to visualize multiple years of the same stock on one chart to visualize annual patterns that occur on a given stock.

Why we selected the topic:
We are casual investors and thought these features would be useful to us when we are doing research on different stocks and cryptocurrencies to purchase.
